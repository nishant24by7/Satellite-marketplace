# 💸 Simple Payment dApp (Soroban on Stellar)

## 📌 Project Description
Simple Payment dApp is a decentralized application built using Soroban smart contracts on the Stellar network. It enables users to send XLM (Stellar Lumens) to any wallet address in a secure, fast, and decentralized way.

This project focuses on simplifying peer-to-peer crypto payments with minimal UI and seamless blockchain interaction.

---

## ⚙️ What It Does
- Accepts recipient wallet address
- Accepts amount of XLM to send
- Verifies sender authorization
- Executes on-chain XLM transfer
- Emits event logs for transparency and tracking

---

## ✨ Features
- 🔐 Secure Transfers with Wallet Authentication  
- ⚡ Instant Payments on Stellar Network  
- 🌍 Send XLM to Any Valid Address  
- 🧾 On-chain Event Logging  
- 🪶 Lightweight & Gas Efficient  
- 🔌 Easy Integration with Frontend Apps  

---

## 🛠️ Tech Stack
- Soroban (Stellar Smart Contracts)
- Rust
- Stellar SDK
- Optional Frontend: React / Next.js

---

## 🚀 How It Works
1. User connects wallet (Freighter / Albedo)
2. Inputs:
   - Recipient Address
   - Amount (XLM)
3. Smart contract:
   - Verifies sender (`require_auth`)
   - Transfers XLM via token contract
4. Emits event for record keeping
